**GENERAL USAGE NOTES**

---

**КРАТКОЕ ОПИСАНИЕ ПРОГРАММЫ**

---

Данная программа предназначеная для определения рейтинга копьютерных игр 
на основании законов Российской Федерации.

Данная программа представлена в виде тестирования.

По ответам на вопросы пользователя программа определяет один из следующих рейтингов игры:

- 0+;

- 6+;

- 12+;

- 16+;

- 18+.



**Installing**

---

**Установка и запуск программы**

---

Данная программа написана с помощью языка JavaScript.

Компиляция не требуется.

Установка не требуется.

Чтобы запустить программу, необходимо открыть файл index.html с помощью одного из следующих браузеров 
[указаны версии браузеров, в которых код ТОЧНО будет работать корректно, версии, ниже представленных, не проверены]:

* «Opera» версии не ниже 41.0;

* «Яндекс.Браузер» версии не ниже 16.10;

* «Google Chrome» версии не ниже 54.0;

* «Mozilla Firefox» версии не ниже 49.0;

* «Internet Explorer» версии не ниже 10;

* «Safari» версии не ниже 5.



Прочее

---

В папке tests представлено тестирование объектов QuestionBank и Testing с помощью библиотеки QUnit.

Для запуска тестов достаточно открыть в браузере файл QBtest.html или файл Testingtest.html.
